# Classroom Online Portal

Simple interactive website with separate teacher and student workflows:

- Teacher login (`teacher` / `admin123`) to upload notes/tests/updates (PDF only) to an individual student.
- Student login by ID + name to view assigned PDFs.
- Student can upload a PDF answer sheet against a test.
- Student can ask doubts; teacher can reply from teacher dashboard.
- Data is stored in browser `localStorage` for demo use.

## Run

Open `index.html` in any modern browser.
